# Missouri Privacy Policy

## Summary

Missouri stores page annotations locally in the browser profile and does not send annotation data to a remote service by default.

## Data Missouri Stores

- Annotation content the user creates, including notes, colors, labels, and placement metadata
- Page identifiers needed to restore annotations when the same page is revisited
- Local extension settings such as overlay preferences

## Where Data Lives

- Annotation and settings data are stored in `chrome.storage.local`
- Data remains on the local machine unless the user exports it or exposes it through another tool


## Data Sharing

Missouri does not sell data, use advertising trackers, or transmit annotation content to a third-party backend as part of normal operation.


## User Control

Users can:

- Delete individual annotations
- Clear annotations from a page
- Remove the extension to stop future collection and storage

## Security Notes

- Missouri uses local browser storage for persistence
